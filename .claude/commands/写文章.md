用户想写一篇博客文章，主题是：$ARGUMENTS

## 完整工作流，严格按顺序执行

### 第一步：收集资料

用 WebSearch 搜索至少 3 个角度的资料，关键词要带当前年份（2026），确保信息是最新的。
对重要的页面用 WebFetch 深入抓取细节。
目标：掌握这个主题的背景、所有方法/功能、最新动态、常见问题。

### 第二步：写文章

根据收集到的资料，用中文写一篇完整的博客文章，要求：
- 零基础友好，但内容深度足够
- 覆盖所有主要方法和功能，不要遗漏
- 结构清晰：用 ## 分章，重要内容用表格/代码块整理
- 长度充足，宁多勿少
- Markdown 格式
- **禁止使用任何 emoji 符号**（包括 ❌ ✅ 🔧 📦 等），用纯文字替代（如"避免"/"推荐"/"×"/"√"）
- **提示词、命令、示例代码一律用代码块（```）包裹**，博客会自动为代码块添加复制按钮

### 第三步：从相册随机取一张封面图

用以下 Node.js 脚本从 Supabase photos 表取图片列表，随机选一张作为封面：

```js
    });
  });
```

### 第四步：发布文章到 Supabase

把文章写入 posts 表，字段说明：
- `title`：文章标题，简洁吸引人
- `slug`：URL 友好的英文 slug，用连字符，比如 `how-to-use-codex-2026`
- `description`：50-80 字摘要，用于列表页预览
- `content`：完整 Markdown 正文
- `status`：`"published"`
- `category`：从 `tech` / `life` / `thoughts` / `design` 中选一个
- `tags`：相关标签数组，3-5 个
- `cover_image`：上一步选出的相册图片 URL
- `is_pinned`：`false`
- `created_at` / `updated_at`：`new Date().toISOString()`

```js
// 发布脚本模板
const https = require('https');
const SUPABASE_URL = 'https://vbaoppgofsjbquwegyjb.supabase.co';
const KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6InZiYW9wcGdvZnNqYnF1d2VneWpiIiwicm9sZSI6InNlcnZpY2Vfcm9sZSIsImlhdCI6MTc2NjIyNzU5MCwiZXhwIjoyMDgxODAzNTkwfQ.nvgU0bRWDpM7TvRDIWxo__8yXFEZNva22yl_tJlWRNQ';

const now = new Date().toISOString();
const payload = JSON.stringify({
  title: '文章标题',
  slug: 'article-slug',
  description: '文章摘要',
  content: '正文内容',
  status: 'published',
  category: 'tech',
  tags: ['标签1', '标签2'],
  cover_image: '相册图片URL',
  is_pinned: false,
  created_at: now,
  updated_at: now
});

const req = https.request(
  new URL(SUPABASE_URL + '/rest/v1/posts'),
  {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'apikey': KEY,
      'Authorization': 'Bearer ' + KEY,
      'Prefer': 'return=representation'
    }
  },
  (res) => {
    let d = '';
    res.on('data', c => d += c);
    res.on('end', () => {
      const post = JSON.parse(d)[0];
      console.log('✅ 发布成功');
      console.log('ID:', post.id);
      console.log('访问路径: /blog/' + post.slug);
      console.log('封面:', post.cover_image);
    });
  }
);
req.write(payload);
req.end();
```

### 第五步：告知用户结果

输出：
- 文章标题
- 访问地址：`/blog/<slug>`
- 使用的封面图 URL
- 字数统计

## 注意事项

- 封面图**必须**从相册取，禁止用 Unsplash 或其他外部图片
- 文章**必须**是 published 状态（不是草稿）
- slug 必须是英文小写+连字符，不能有空格或中文
- 搜索资料时关键词加年份「2026」确保信息最新
- 谷歌官方 APK / 受版权保护的文件不要直接托管，链接到 APKMirror 页面即可
